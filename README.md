Функционал:

0. Регистрация
1. аутентификация
2. авторизация JWT
3. Календарь
4. Возможность размещать информацию о новых мероприятиях, включаю фото, аудио или видео.
5. Возможность поставить напоминание.
6. Возможность сортировки мероприятий по темам.



Возможно будет реализовано)
