# NodeJS



## Постановка задачи:  
Создать приложения, в котором можна вести свой блог
  


## Функционал: 
### Обязательный
1. Регистрация(через электронную почту)
2. аутентификация
3. авторизация JWT
4. Создание профиля(Ник, фото)
5. Возможность размещать пост.
